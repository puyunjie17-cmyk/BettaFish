# StoryForge v1.0.1

从创意生成商业网文策划、IP 圣经、长篇大纲、网文正文、AI 漫剧剧本、分镜脚本与全链路质检的静态创作工作台。

## 本次启动修复

- 移除 Base64 分片、浏览器端解压和 `document.write` 重建启动器。
- 改为标准 `index.html + assets/app.css + assets/app.js` 静态结构。
- 本地存储不可用时自动切换到会话内存，不再中断启动。
- 启动失败时显示真实错误，例如 `assets/app.js` 未上传。
- 生成完成后即时更新七阶段完成状态。

## 使用

正式 AI 模式直接打开站点。演示模式在地址末尾加 `?mock=1`，不会调用真实模型。

项目默认存于浏览器。隐私模式或内置浏览器禁止本地存储时，项目只保留在当前页面会话，必须及时导出 JSON。
